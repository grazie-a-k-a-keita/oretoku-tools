## OretokuTools

主にデザイン、開発に役立つサイト、ツールの寄せ集めサイト

### Develop

- 開発の始め方

```bash
# Server Start
npm run dev
```

- デプロイ方法

`develop` ブランチで開発を行い、`main` ブランチにマージすることで、[Vercel](https://vercel.com/) 上に自動デプロイされる
